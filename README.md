## ğŸš€ Deployment Instructions

### Step 1: Create GitHub Repository

1. Go to https://github.com and sign in (or create an account)
2. Click "New Repository"
3. Name it `lenderpepe` or `lender-pepe-website`
4. Make it **Public**
5. Click "Create Repository"

### Step 2: Upload Files

**Option A: Drag & Drop (Easiest)**
1. In your new repository, click "uploading an existing file"
2. Drag all files from this folder into the browser
3. Click "Commit changes"

**Option B: Command Line**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/lenderpepe.git
git push -u origin main
```

### Step 3: Deploy to Vercel

1. Go to https://vercel.com and sign in with GitHub
2. Click "Add New Project"
3. Import your `lenderpepe` repository
4. Keep default settings and click "Deploy"
5. You'll get a `.vercel.app` URL

### Step 4: Connect Custom Domain (www.lenderpepe.com)

1. In Vercel dashboard, go to your project â†’ Settings â†’ Domains
2. Add `www.lenderpepe.com`
3. Vercel will show DNS records to add

### Step 5: Configure NameSilo DNS

1. Log into https://www.namesilo.com
2. Go to Domain Manager â†’ Find `lenderpepe.com`
3. Click the blue globe icon (DNS Manager)
4. Delete any existing `www` records
5. Add these records:

| Type | Host | Value | TTL |
|------|------|-------|-----|
| A | www | 76.76.21.21 | 3600 |
| CNAME | www | cname.vercel-dns.com | 3600 |

6. Click Submit

### Step 6: Wait for SSL

- SSL certificate will be auto-generated by Vercel (5-60 minutes)
- DNS propagation: 15 min - 2 hours
